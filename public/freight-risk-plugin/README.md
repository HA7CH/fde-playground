# 货代对单/做账防错插件

这是第一版轻量旁路原型，用于验证两类核心场景：

- 对单防错：比对 SI/补料、HBL、MBL 的关键字段，输出红黄灯差异报告。
- 做账防漏：检查签费、USD/RMB DN、供应商账单、操作备注中的疑似漏收、漏录、未确认费用。

## 使用方式

直接用浏览器打开：

`index.html`

不需要安装依赖，也不需要启动服务。

## 当前边界

- 不替换现有货代系统。
- 不自动提交船司。
- 不自动入账。
- 不自动放单。
- 不承诺 AI 结果 100% 正确。
- 每条风险都应由跟单、业务、财务或主管最终确认。

## 第一版适合验证什么

1. 历史错单能否被差异报告抓出来。
2. 合法差异是否会造成过多误报。
3. 做账漏费规则是否能发现真实漏收、漏录风险。
4. 一线是否愿意每天使用。
